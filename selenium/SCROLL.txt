SCROLL FUNCTIONS

THIS SCROLL FUNCTION CAN BE PERFORMED BY USING A FUNCTION CALLED JAVA EXECUTOR.
THIS JAVA EXECUTOR PERFORMS THE OUTER LAYER FUNCTIONALITY ON A WINDOW

SYNTAX FOR JAVA EXECUTER IS

JAVAEXECUTOR VARIABLE_NAME= (JAVAEXECUTOR)D;

THERE ARE 3 WAYS WHERE WE CAN USE THIS SCROLL FUNCTIONS:
1.  SCROLL DOWN BY USING PIXELS

2.  SCROLL UP  BY USING PIXELS

3.  SCROLL DOWN BY THE VISIBILITY OF AN ELEMENT.


SYNTAX FOR SCROLL DOWN:
JAVASCRIPTEXECUTOR JS=(JAVAESCRIPTXECUTOR)D;
JS.EXECUTESCRIPT("WINDOW.SCROLLBY(0,1000)"," ");

SYNTAX FOR SCROLL UP:
JAVASCRIPTEXECUTOR JS1=(JAVASCRIPTEXECUTOR)D;
JS1.EXECUTESCRIPT("WINDOW.SCROLLBY(0,-600)"," ");
